# ComfyUI-BAIBAOXIANG

一个面向日常生产流程的 ComfyUI 百宝箱插件。  
核心思路很简单: 把我平时高频使用的小工具节点汇聚到一起，减少重复搭图和手工操作。

## 插件定位

- 面向高频、实用的小功能节点集合
- 以“能直接提效”为第一目标
- 覆盖文本处理、批量提交、图像拼接等常见场景

## 当前包含的节点

1. `KK_TimeString`（输出时间字符串）
- 按需组合年/月/日/时/分/秒
- 支持前缀和后缀
- 适合做自动命名、批量输出标记

2. `KK_TextToList`（文本转字符串列表）
- 将文本解析为字符串列表
- 支持 JSON / Python 字面量格式
- 适合和批量节点联动

3. `KK_ImageStitching`（图像自动拼接）
- 支持横向/竖向拼接
- 支持统一尺寸、间距、间距颜色
- 支持数量限制（按列/按行分组）

4. `KK_AutoBatchSubmit`（自动批量提交任务）
- 可按文件夹图片批次、字符串批次进行任务拆分
- 支持自动向 ComfyUI 队列提交后续任务
- 适合一键跑完整批次

5. `KK_KeywordImageBatch`（根据关键词输出对应图片-自动批量提交任务）
- 从文本中匹配关键词
- 从指定文件夹中匹配对应图片并输出（最多 6 张）
- 可与自动批量提交联动，适合“文案 + 配图”类流程

## 安装方式

1. 将本项目放入 ComfyUI 的 `custom_nodes` 目录  
目录示例: `ComfyUI/custom_nodes/ComfyUI-BAIBAOXIANG`
2. 重启 ComfyUI

## 快速使用建议

1. 文本批处理  
`文本转字符串列表 -> 自动批量提交任务`  
用于批量 prompt 或批量文案任务。

2. 关键词配图批处理  
`文本转字符串列表 -> 根据关键词输出对应图片-自动批量提交任务`  
用于根据文本自动挂载对应素材图。

3. 结果拼图  
`图像自动拼接`  
用于批量结果预览、对比图输出、联系单风格导出。

## 使用注意

- 批量提交节点依赖 ComfyUI 本地接口（`/prompt`），请确保节点运行在当前 ComfyUI 实例内。
- 涉及文件夹输入时，使用本地绝对路径，避免路径解析问题。
- `最大提交数` 会与实际可用数据长度共同决定最终批次数。
